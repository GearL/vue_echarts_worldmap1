# worldareamap
### 此世界地图根据颜色深浅来显示全球地区数据多少
### 功能：
####     1.可点击图斑
####     2.可自适应大小
### 注意：
####     1.此地图数据阈值固定
####     2.此地图容器由id选中，故而同一个页面只能出现一次，可用ref来解决
### 数据：
